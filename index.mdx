## Platform Overview

DormWay helps students navigate campus life with intelligent scheduling, task management, and personalized insights.

### Core Services

| Service | Port | Purpose |
|---------|------|---------|
| **API Router** | 3001 | Express.js backend, BFF endpoints |
| **Engine** | 3030 | Temporal workflows, data processing |
| **Zuplo** | 9000 | API Gateway, auth, rate limiting |
| **LockedIn** | 3008 | Next.js student web app |
| **Admin** | 3004 | Internal admin dashboard |

## Getting Started

<Steps>
  <Step title="Clone the repos">
    ```bash
    cd dormway-beta-workspace
    make bootstrap
    ```
  </Step>
  <Step title="Configure Doppler">
    ```bash
    cd .repos/dormway-platform
    doppler login && doppler setup
    ```
  </Step>
  <Step title="Start services">
    ```bash
    make dev
    make status  # verify health
    ```
  </Step>
</Steps>
