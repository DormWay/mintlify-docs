## Analysis Template

When creating new analyses, include:
1. **Context**: What prompted this analysis
2. **Current State**: What exists today
3. **Gaps/Issues**: What's missing or broken
4. **Recommendations**: Proposed solutions or approaches
5. **Next Steps**: Action items and ownership
6. **References**: Related docs and data sources
