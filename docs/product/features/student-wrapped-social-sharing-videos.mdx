# Student Wrapped - Social Sharing Videos

**Status:** Concept/Planning
**Tech Stack:** React Remotion (video rendering)
**Goal:** Create compelling "Spotify Wrapped" style videos showcasing student accomplishments for social sharing

---

## Executive Summary

Create shareable video content that celebrates student accomplishments using the rich data we collect across BrainGains, SyllabusProcessor, SemesterProcessor, and DayPlan systems. These videos should be:
- **Visually compelling** - Fast-paced, animated, celebration-focused
- **Shareable** - Optimized for Instagram Stories, TikTok, Twitter/X
- **Timely** - Released at strategic moments to saturate social media
- **Personal** - Unique to each student with their actual data

---

## Data Sources

### 1. BrainGains (Document Activity)
- **Documents uploaded**: Total count
- **Pages processed**: Total page count across all documents
- **Query activity**: Number of queries run against knowledge base
- **Document types**: Breakdown by notes, assignments, syllabi, readings
- **Upload timeline**: When documents were added (study habit patterns)
- **Most active upload days**: Peak engagement moments

### 2. SyllabusProcessor (Course Structure)
- **Courses enrolled**: Total course count
- **Assignment count**: Total assignments tracked
- **Estimated workload**: Total hours committed
- **Grade weight distribution**: How grades are calculated across courses
- **Course diversity**: Departments, majors represented
- **Strictest policies**: Most challenging attendance/late policies

### 3. SemesterProcessor (Semester Analytics)
- **Peak weeks**: Number of high-intensity weeks navigated
- **Conflicts resolved**: Schedule and assignment conflicts handled
- **Learning synergies**: Cross-course connections identified
- **Workload intensity**: Average assignments per week
- **Early warnings**: Potential issues flagged and addressed
- **Study time allocation**: Hours across different courses

### 4. DayPlan (Daily Execution)
- **Days planned**: Total days with active plans
- **Schedule adherence**: Percentage of time following plan
- **Study blocks completed**: Scheduled study sessions
- **Busiest day**: Highest intensity day
- **Energy optimization**: Best study times identified
- **Deviations handled**: Adaptations to schedule changes

## Release Timing Strategy

### Primary Release Moments (Maximum Social Saturation)

#### 1. **End of Semester** - "Semester Wrapped"
**Timing:** Final exam week OR day after final grades posted
**Why:**
- Students have free time (exams done or grades in)
- High emotional investment (relief, celebration, reflection)
- Natural sharing moment (bragging rights, parent updates)
- Concentrated timing = social feed saturation
- Everyone gets theirs at same time = FOMO effect

**Trigger:**
- Option A: 2 days before first final exam (during study period)
- Option B: Day after final grades posted to university registrar
- Recommendation: **Option A** - Stress relief moment, study break content

**Notification:**
- Push: "Your Fall 2025 Semester Wrapped is ready! ğŸ‰"
- Email: "You made it through another semester - see how far you came"

---

#### 2. **Mid-Semester** - "Halfway There Check-In"

**Why:**
- Motivation boost during slump period
- Progress reinforcement
- Re-engagement with app
- Natural checkpoint

## Social Media Optimization

### Platform Specifications

#### Instagram Stories / Reels
- **Aspect Ratio:** 9:16 (1080x1920)
- **Duration:** 60-90 seconds max
- **Format:** MP4 (H.264)
- **Captions:** Always include (for sound-off viewing)
- **Call-to-action:** "Share to your story" button

#### TikTok
- **Aspect Ratio:** 9:16 (1080x1920)
- **Duration:** 30-60 seconds (shorter = better)
- **Format:** MP4
- **Hashtags:** Auto-generate based on school, semester
- **Trending sounds:** Consider integration

#### Twitter/X
- **Aspect Ratio:** 16:9 (1920x1080) OR 9:16
- **Duration:** 45 seconds max
- **Format:** MP4
- **File size:** Under 512MB
- **Suggested tweet:** Pre-populate with stats

#### YouTube Shorts
- **Aspect Ratio:** 9:16
- **Duration:** 60 seconds max
- **Format:** MP4
- **Title template:** "My [SEMESTER] at [SCHOOL] - DormWay Wrapped"

### Data Enrichment Pipelines

#### Pipeline 1: Semester Wrap Preparation
**Trigger:** 1 week before semester end
**Process:**
1. Query all user activity from semester start to current date
2. Calculate aggregate statistics
3. Identify peak weeks from workload analysis
4. Determine learning style/persona
5. Generate comparative stats (vs. average, vs. previous semester)
6. Store in `service_data` table with method='semester_wrap_data'

**Storage Schema:**
```typescript
{
  method: 'semester_wrap_data',
  data: {
    term: string,          // "Fall 2024"
    stats: StudentStats,
    comparative: ComparativeStats,
    persona: StudyPersona,
    temporal: TemporalInsights,
    generatedAt: string
  }
}
```

---

#### Pipeline 2: Video Generation Queue
**Trigger:** Release timing hit (e.g., 2 days before finals)
**Process:**
1. Fetch pre-computed wrap data from `service_data`
2. Enqueue video generation job for each active user
3. Process via React Remotion rendering service
4. Store video in S3
5. Create notification with video link

**Queue Structure:**
```typescript
interface VideoRenderJob {
  userId: string
  term: string
  videoType: 'semester_wrapped' | 'midterm_checkin' | 'year_review'
  data: StudentStats
  customization: {
    schoolColors?: [string, string]
    musicEnabled: boolean
    skipStats?: string[] // privacy
  }
  outputFormat: {
    platform: 'instagram' | 'tiktok' | 'twitter' | 'youtube'
    aspectRatio: '9:16' | '16:9'
    duration: number // seconds
  }
}
```

## Privacy & Ethics Considerations

### 1. **Opt-Out Mechanism**
- User settings: "Disable Wrapped videos" toggle
- Honor opt-out before generating/notifying

### 2. **Data Sensitivity**
- Don't reveal actual grades (we don't have them anyway)
- Don't share stress/mental health inferences
- Avoid language that could make struggling students feel worse
- Positive framing: focus on effort, not outcomes

### 3. **Customization**
- Let users hide specific stats before sharing
- Option to regenerate with different stats
- Delete video permanently

### 4. **Consent**
- First-time: Explain what Wrapped is, get consent
- Remind: "This will be shared publicly if you post it"
