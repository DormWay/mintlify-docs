## Implementation Architecture

### 1. QR Code Generation
```swift
// In DormWay Desktop/Mobile
func createStudyGroup() -> StudyGroupSession {
    let session = StudyGroupSession(
        id: UUID(),
        hostId: currentUser.id,
        courseName: detectCurrentCourse(),
        expiresAt: Date().addingHours(24)
    )
    
    // Generate join URL
    let joinURL = "https://study.dormway.app/join/\(session.shortCode)"
    
    // Create QR with branding
    return QRCode(
        url: joinURL,
        logo: "DormWay",
        caption: "Scan to join StudyGroup"
    )
}
```

### 2. Web App (study.dormway.app)
Lightweight React app that:
- Requires ZERO authentication to join
- Opens immediately to collaborative editor
- Shows who's in the room (anonymous animals if not signed in)
- Prompts to save work by creating DormWay account

### 3. Smart Routing
```javascript
// When QR code is scanned
if (hasDormWayApp()) {
    // Deep link to native app
    window.location = "dormway://studygroup/join/" + roomCode
} else if (isMobile()) {
    // Open web app with mobile UI
    redirectTo("/mobile-editor/" + roomCode)
} else {
    // Full web editor
    redirectTo("/editor/" + roomCode)
}
```
