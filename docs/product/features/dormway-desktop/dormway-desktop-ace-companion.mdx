# DormWay Desktop - Ace Companion

## Overview

A focused macOS desktop application that serves as a powerful companion to the DormWay mobile app, specifically designed to enhance the Ace (BrainGains) experience for students during study sessions and classes.

**Core Value Proposition**: Seamlessly add documents and notes to Ace while studying on your Mac, with real-time collaboration features for shared class notes.

## Target Users

- Students who use MacBooks for studying and note-taking
- Students in lecture halls who want to quickly add class materials to Ace
- Study groups who want to collaborate on notes in real-time
- Power users who want faster desktop access to Ace search

## User Experience Flow

### First Launch
1. Download from Mac App Store
2. Sign in with existing DormWay account
3. Grant necessary permissions (files, notifications)
4. Set up global hotkeys
5. Tutorial showing key features

### Daily Use
1. **Morning**: Check Ace for today's class prep
2. **During Class**: 
   - Join notes room automatically (location-based)
   - Drag lecture slides into Ace
   - Collaborate on notes with classmates
3. **Study Time**: 
   - Quick search for concepts
   - Add study materials
   - Create summary notes
4. **Evening**: Export day's notes for review

## Technical Requirements

### System Requirements
- macOS 14.0+ (Sonoma)
- Apple Silicon or Intel Mac
- 200MB disk space
- Internet connection for sync

### Development Stack
- SwiftUI for macOS
- Shared Swift packages from iOS
- AppKit for system integration
- Combine for reactive programming
- Core Data for local storage

## Privacy & Security

- All notes encrypted in transit and at rest
- Optional end-to-end encryption for notes rooms
- No tracking of personal study habits
- Local-first architecture with sync
- Compliant with university data policies

## Performance Targets

- App launch: `<1 second`
- Quick note window: `<200ms`
- Search results: `<500ms`
- Document upload: Background with progress
- Real-time sync latency: `<100ms`

## Success Metrics

## Conclusion

DormWay Desktop focuses on enhancing the Ace experience for desktop users without trying to replicate the full mobile app. By leveraging 70%+ of existing iOS code and focusing on desktop-specific workflows, we can deliver a powerful companion app that makes DormWay indispensable for serious students.

The real-time collaboration features could become a unique differentiator, making DormWay the default way students share and access class notes.
