# User Story - The Laptop Pass Problem

## Current Reality: A Day in the Life

### 3:00 PM - Library Study Room

**Sarah, Mike, Emma, and Jake** meet for their EECS 281 group project.

**The Chaos Begins**:

```
Sarah: "Ok, I'll start the doc on my laptop"
*types introduction*

Mike: "Wait, let me add the algorithm part"
*Sarah saves, emails link, Mike opens*
"Ugh, it's view-only"

Sarah: "Here, just use my laptop"
*physically passes MacBook*
*Sarah now can't reference her notes*

Emma: "Can someone share screen so I can see?"
*awkward huddle around one screen*

Jake: "I'll just write my part separately and we'll merge later"
*Version chaos begins*
```

### 4:30 PM - The Merge Nightmare

```
Sarah: "Ok everyone email me your parts"
*receives 4 different documents*
*formatting is all different*
*duplicate sections*
*missing transitions*

Mike: "Wait, I updated mine, sending v2"
*now 5 documents*

Emma: "Which algorithm explanation are we using?"
*comparing 3 versions side-by-side*

Jake: "I think we lost the intro Sarah wrote"
*Sarah's laptop was closed, changes lost*
```

### 6:00 PM - Submission Panic

```
Final document: FinalProject_v5_FINAL_ACTUALLY_FINAL_2.docx
- Inconsistent formatting
- Repeated concepts
- Missing Jake's conclusion
- No idea who wrote what for peer evaluation
```
