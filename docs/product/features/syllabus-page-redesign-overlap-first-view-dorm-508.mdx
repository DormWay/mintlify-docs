### **Phase 1: Data Layer (6 hours)**

**New Hook:** `useOverlapAnalysis.ts`

**Purpose:** Aggregate and analyze all deadlines to detect overlap periods.

**Key Data Structures:**

```typescript
interface OverlapPeriod {
  weekKey: string;          // "2025-W45"
  weekStart: string;        // ISO date
  weekEnd: string;
  weekLabel: string;        // "Nov 4-10"
  intensity: number;        // 1-10 scale
  totalEstimatedHours: number;
  overlapCount: number;     // Number of items on same day

  assignments: Assignment[];
  exams: Assignment[];

  // Critical days (3+ items)
  criticalDays: {
    date: string;
    dayLabel: string;
    items: Assignment[];
    totalHours: number;
  }[];

  affectedCourses: {
    courseId: string;
    courseName: string;
    itemCount: number;
  }[];

  recommendations: {
    type: 'start_early' | 'request_extension' | 'block_time';
    priority: 'high' | 'medium' | 'low';
    action: string;
  }[];

  severity: 'critical' | 'high' | 'moderate' | 'low';
}
```

**Algorithm:**

1. Group all assignments by week
2. Identify weeks with 2+ major items (exams, papers, projects)
3. Calculate intensity score based on:
   - Total estimated hours
   - Number of exams
   - Same-day conflicts
4. Generate AI recommendations:
   - Start early for exam-heavy weeks
   - Block study time before critical days
   - Suggest extensions when papers overlap with exams
5. Classify severity and sort by criticality

**Data Sources:**
- `summary.weekly_cadence` - Workload per week
- `summary.conflictAnalysis.conflicts` - Detected conflicts
- `summary.course_assignments` - All assignments from Canvas/syllabi
- `syllabusCourses` - Course syllabus data
- `flattenedAssignments` - Collected from `collectCourseAssignments()`

### Technical Details

**Temporal Context Calculation Logic:**
```typescript
const now = new Date();  // Ground in today's date
const termStart = new Date(startDate);
const termEnd = new Date(endDate);
const totalMs = termEnd.getTime() - termStart.getTime();
const elapsedMs = now.getTime() - termStart.getTime();
const weeksElapsed = Math.max(0, Math.floor(elapsedMs / (7 * 24 * 60 * 60 * 1000)));
const totalWeeks = Math.max(1, Math.ceil(totalMs / (7 * 24 * 60 * 60 * 1000)));
const currentWeek = Math.min(weeksElapsed + 1, totalWeeks); // 1-indexed
```
