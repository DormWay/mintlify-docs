# Features Documentation

## Purpose
This section contains feature specifications, implementation documentation, and status tracking for all DormWay platform features across web, iOS, and backend systems.

## Structure
Features are organized by feature area, with each major feature having its own subdirectory containing specifications, implementation plans, and related documentation.

3. **Planning**: Create work plan in [Workplan](/docs/engineering/ai-crews/readme)
4. **Implementation**: Track progress here in Features/
5. **Launch**: Update status to Active upon release
