# LockedIn UX Specification - Navigation Consolidation

> Canonical nav truth lives in [Navigation IA](/docs/engineering/architecture/sot-navigation-ia). This spec mirrors it for UX context.

## Executive Summary

- **Goal:** Consolidate LockedIn navigation from ~12 items to 5 primary items
- **Key Change:** Library is the unified surface for files + notes (no standalone Notes tab)
- **Key Change:** Settings moves out of the main nav (top-right access)
- **Philosophy:** Focus on crisis-prevention core, deprioritize redundant features
- **Timeline:** 4-phase rollout (Foundation â†’ Sidebar â†’ Pages â†’ Green Mode)

## Design Philosophy

**Core Principle**: Students should feel ownership over their academic life through direct manipulation, with DormWay serving as a command centerâ€”not a productivity carnival.

### Why This Change

| Current Problem | Solution |
|-----------------|----------|
| 12+ nav items overwhelm users | 5 focused items |
| Notes Hub competes with Notion/Docs | Deprioritize (per-course access) |
| "Inbox" mental model is confusing | Merge into Today dashboard |
| AI features mixed with non-AI | Clear AI mode separation |
| No eco-conscious option | Green Mode for AI-conscious users |

---

## Current Feature Analysis

### Notes (Library) - KEEP (inside Library)

**What it does:**
- Full note-taking with per-course organization
- Auto-detects current class based on schedule
- Autosave with cloud sync
- Export to markdown
- Course association with smart title prefixing

**Usage indicators (from telemetry):**
- Tracks: `note_opened`, `note_created`, `note_autosaved`, `note_exported`, `note_deleted`
- Deep integration with courses system
- Context-aware (knows which class you're in)

**Verdict:** Valuable feature but not core to crisis-prevention value prop. Students use Notion/Google Docs for notes. Keep accessible via Library with course filters; do not give a standalone sidebar entry.

### Inbox - MERGE into Today

**What it does:**
- Shows items due within 48 hours
- Study suggestions based on syllabus data
- Quick captures (thoughts saved during focus sessions)
- Start focus sessions from items
- Plan study blocks
- School Mode integration (pauses during focus)

**Usage indicators:**
- Integrated with planner blocks
- Focus session triggers
- Full telemetry coverage

**Verdict:** This is actually a focus/action hub, not an "inbox" in traditional sense. The mental model of "inbox" is confusing. Merge functionality into Today dashboard.
