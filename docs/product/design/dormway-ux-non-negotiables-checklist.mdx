# DormWay UX Non-Negotiables Checklist

One-page checklist for quick reviews. Source of truth is [DormWay UX Non-Negotiables (Gen Z)](/docs/product/design/dormway-ux-non-negotiables-gen-z).

## Control and Clarity
- [ ] Today shows next action above the fold.
- [ ] A finite, clearable state exists ("all set").
- [ ] No infinite scroll feed in productivity surfaces.
- [ ] Primary actions are one tap away.

## Trust and Transparency
- [ ] Sync freshness visible on key surfaces.
- [ ] Data sources are visible where needed.
- [ ] No dark patterns or surprise state changes.

## Capture Speed
- [ ] Task capture can be done in under 3 seconds.
- [ ] Camera/NLP capture path exists or is planned.

## Platform Fit
- [ ] iPhone: one-handed, thumb-reachable controls.
- [ ] iPad: multi-column layout for planning.
- [ ] Mac: keyboard-first navigation and shortcuts.

## AI and Assistance
- [ ] AI reduces work invisibly (no theater).
- [ ] AI actions are optional and reversible.

## Notifications
- [ ] Notifications are high-signal only.
- [ ] No guilt badges or spammy reminders.

## Social Presence
- [ ] Presence is passive and opt-in.
- [ ] No social feed or public activity timeline.
