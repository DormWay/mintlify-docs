## Design Philosophy

**Core Principles**:
1. **Information Density**: Maximum info, minimal clicks
2. **Real-Time Feedback**: Live updates without refresh
3. **Quick Actions**: Common tasks one click away
4. **Error Prevention**: Clear warnings before destructive actions
5. **Progressive Disclosure**: Show details on demand

**Design Language**:
- Dark theme by default (developer-friendly)
- Monospace fonts for code/logs
- Status colors: Green (ok), Yellow (warning), Red (error), Gray (disabled)
- Glassmorphic cards with subtle shadows
- Smooth animations (120fps)

## Accessibility

**WCAG 2.1 Level AA Compliance**:
- Keyboard navigation support
- ARIA labels for screen readers
- Sufficient color contrast (4.5:1 minimum)
- Focus indicators on all interactive elements
- Alt text for all icons

**Keyboard Shortcuts**:
- `Cmd/Ctrl + K`: Quick search
- `Cmd/Ctrl + R`: Refresh current page
- `Cmd/Ctrl + L`: Open logs viewer
- `Cmd/Ctrl + D`: Open diagnostics
- `Esc`: Close modals/panels
- `Tab`: Navigate between elements
- `Space`: Toggle checkboxes
