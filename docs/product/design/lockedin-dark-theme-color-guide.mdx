# LockedIn Dark Theme Color Guide

## Overview

This document provides a comprehensive reference for all colors used in the LockedIn dark theme. The dark theme uses a deep purple/indigo gradient background with glassmorphic UI elements and subtle accent colors.

## ğŸ¯ Accessibility Notes

- **WCAG AA Compliant**: All text colors meet minimum contrast ratios on dark backgrounds
- **Primary text** (`#f1f5f9`) has >7:1 contrast on `rgba(9, 2, 22, 0.92)` backgrounds
- **Secondary text** (`#cbd5e1`) maintains >4.5:1 contrast
- **Minimum readable text** is `#64748b` which meets 3:1 for large text

## ğŸ“ Notes

- Dark theme uses **opacity-based layering** for depth (`0.55`, `0.92`, `0.98`)
- **Glassmorphic effects** rely on `backdrop-filter: blur()` with translucent backgrounds
- **Brand gradients** use 135deg angle for consistency
- **Purple family** (`#8b5cf6`, `#6d28d9`, `#a78bfa`) is the primary accent spectrum
- **Time-based gradients** provide contextual atmosphere throughout the day
