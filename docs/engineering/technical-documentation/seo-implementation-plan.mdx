# SEO Implementation Plan for DormWay LockedIn

## Executive Summary
This document outlines the comprehensive SEO strategy and implementation plan for the DormWay LockedIn web application to improve organic search visibility, drive qualified traffic, and support the PLG motion.

---

## Current State Analysis

### Strengths
- Strong brand messaging ("The OS for College Life")
- School detection and personalization
- Dynamic content with syllabus previews
- Mobile-responsive design

### Gaps Identified
1. **Missing Meta Tags** - No OpenGraph or Twitter cards
2. **No Structured Data** - Missing schema.org markup
3. **Limited Page Metadata** - Basic title/description only
4. **No Sitemap** - Search engines can't efficiently crawl
5. **Performance Issues** - Large bundle sizes affect Core Web Vitals
6. **No Canonical URLs** - Potential duplicate content issues
7. **Missing Alt Text** - Images lack descriptive text

## Priority 3: Performance Optimization

### 3.1 Core Web Vitals Targets
- **LCP (Largest Contentful Paint)**: < 2.5s
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1

### 3.2 Optimization Tactics
```typescript
// Image optimization
import Image from 'next/image';

// Use Next.js Image for automatic optimization
<Image
  src="/hero-image.webp"
  alt="Student using DormWay to plan semester"
  width={800}
  height={600}
  priority // for above-fold images
  placeholder="blur"
/>

// Font optimization
import { Inter } from 'next/font/google';
const inter = Inter({
  subsets: ['latin'],
  display: 'swap', // Prevent FOIT
});

// Code splitting
const HeavyComponent = dynamic(() => import('./HeavyComponent'), {
  loading: () => <Skeleton />,
  ssr: false,
});
```

### 3.3 Bundle Size Reduction
- Implement route-based code splitting
- Lazy load below-fold components
- Tree-shake unused dependencies
- Use production builds with minification

## Priority 4: Technical Infrastructure

### 4.1 XML Sitemap
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://www.dormway.app/</loc>
    <lastmod>2025-01-23</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://www.dormway.app/demo</loc>
    <lastmod>2025-01-23</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://www.dormway.app/about</loc>
    <lastmod>2025-01-23</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.6</priority>
  </url>
</urlset>
```

### 4.2 Robots.txt
```
User-agent: *
Allow: /
Disallow: /api/
Disallow: /dashboard/
Sitemap: https://www.dormway.app/sitemap.xml
```

### 4.3 Canonical URLs
```typescript
// Prevent duplicate content issues
export const metadata = {
  alternates: {
    canonical: 'https://www.dormway.app',
    languages: {
      'en-US': 'https://www.dormway.app',
    },
  },
};
```
