# SoT - Navigation IA

## Scope
- Cross-platform navigation structure for LockedIn (web) and DormWay (iOS)
- Main navigation items, ordering, and naming conventions
- Settings placement and accessibility patterns
- Focus Panel / Inspector integration with navigation

## Non-goals
- Widget layout within surfaces (see [Home & Widgets](/docs/engineering/architecture/sot-home-widgets))
- Deep linking URL patterns (implementation detail)
- Visual design of navigation components

## Canonical Navigation Structure

### Main Navigation (5 Items)

| Order | Label | Web Route | iOS Tab | Description |
|-------|-------|-----------|---------|-------------|

## Invariants & Contracts

1. **5-item navigation maximum** - Main nav must have exactly 5 items across all platforms
2. **Consistent naming** - Same labels on web and iOS (Today, Tasks, Schedule, Courses, Library)
3. **Settings outside main nav** - Settings accessed via top-right, never as a main nav item
4. **Order consistency** - Same item order across platforms (Today first, Library last)
5. **Focus Panel integration** - All surfaces support inspector/detail panel via URL state

## Focus Panel Integration

The Focus Panel (Inspector) provides contextual details for entities without leaving the current surface.

**URL state note:** Current web uses `?inspector=` in home-v2. Long-term canonical state is `?focus=` (see SoT - Command Search & Focus Panel and [Student OS URL Map](/docs/engineering/architecture/student-os/student-os-url-map)).

### Supported Entity Types
| Entity | URL Pattern | Actions |
|--------|-------------|---------|
| task | `?inspector=task:uuid` | Complete, Edit, Delete |
| assignment | `?inspector=assignment:uuid` | Mark Complete, Open Source |
| course | `?inspector=course:uuid` | View Details, Navigate |
| course_list | `?inspector=course_list:uuid` | View All Courses |
| event | `?inspector=event:uuid` | View Details |

### Focus Panel Behavior
- Opens from any navigation surface
- ESC key to close
- URL state preserves deep links
- Max 3 pinned items
- Keyboard navigation supported
