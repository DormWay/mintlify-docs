## Auth Flow

```
User → Auth0 → JWT → Zuplo Gateway → API Router
                         ↓
                    User enrichment
                         ↓
                    RLS via dormway_api
```

---

## Key Concepts

### Authentication Methods
- **Auth0** - Primary identity provider
- **Clerk** - Alternative auth (migration in progress)
- **Device Keys** - iOS device authentication

### Authorization
- **JWT Claims** - User identity and permissions
- **RLS** - Row-Level Security in PostgreSQL
- **Zuplo** - API gateway auth enforcement
