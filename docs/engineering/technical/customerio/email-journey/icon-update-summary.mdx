## Icon System Specifications

### Technical Implementation
All icons follow these standards:
- **Format**: Inline SVG
- **Viewbox**: `0 0 24 24`
- **Stroke Width**: 2px
- **Style**: Lucide React icon set (stroke-based, rounded)
- **Attributes**: `fill="none"`, `stroke-linecap="round"`, `stroke-linejoin="round"`

### Color Palette
- **Brand Purple**: `#795cf0` - Primary actions, links, academic icons
- **Accent Orange**: `#ff8e4d` - Secondary CTAs, event highlights
- **Success Green**: `#10b981` - Success states, completions
- **Error Red**: `#ef4444` - Urgent items, errors
- **Neutral Gray**: `#999999` - Supporting text, metadata

### Icon Sizing
- **Hero Icons**: 48-64px (large featured icons)
- **Section Headers**: 24px (standard heading icons)
- **Inline Icons**: 16px (text-integrated icons)
- **Stat Cards**: 24px (featured statistics)

## Benefits of Professional Icons

### Brand Consistency
- Matches DormWay web application (Lucide React icons)
- Professional appearance for institutional communications
- Consistent with modern SaaS email design

### Accessibility
- SVG icons can be styled for better contrast
- Screen readers can parse SVG titles/descriptions
- No emoji interpretation issues across platforms

### Performance
- Inline SVG = no external requests
- Smaller file size than emoji alternatives
- Faster rendering in email clients

### Flexibility
- Easy to change colors per theme
- Consistent sizing across all email clients
- Can be animated if needed (limited support)

## Design System Integration

All icons follow the DormWay design system specifications:

**Primary Brand Color**: `#795cf0` (Purple)
- Used for: Academic icons, links, primary actions
- Examples: Books, calendars, user icons

**Secondary Accent**: `#ff8e4d` (Orange)
- Used for: Event highlights, secondary CTAs
- Examples: Event cards, location pins

**Success State**: `#10b981` (Green)
- Used for: Completed actions, success messages
- Examples: Checkmarks, processed confirmations

**Alert State**: `#ef4444` (Red)
- Used for: Urgent items, high priority
- Examples: Alert circles, deadline warnings
