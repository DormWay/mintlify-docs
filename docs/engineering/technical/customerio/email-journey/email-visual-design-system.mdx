# DormWay Customer.io Email Campaign Visual Design System

## Brand Colors (from Design System)

### Primary Palette
- **Brand Purple**: `#795cf0` (Primary actions, headers, links)
- **Purple Dark**: `#522bee` (Pressed states, shadows)
- **Purple Light**: `#9d86f9` (Hover states, highlights)
- **Accent Orange**: `#ff8e4d` (Secondary CTAs, alerts)

### Neutrals
- **Near Black**: `#1a1a1a` (Email background)
- **Grey 900**: `#2a2a2a` (Card backgrounds)
- **Grey 800**: `#333333` (Secondary surfaces)
- **Grey 600**: `#666666` (Secondary text)
- **Grey 400**: `#999999` (Tertiary text)
- **Off White**: `#f5f5f5` (Primary text on dark)
- **White**: `#ffffff` (Headers, emphasized text)

### State Colors
- **Success**: `#10b981` (Green - Success messages)
- **Error**: `#ef4444` (Red - Error states)
- **Warning**: `#fbbf24` (Amber - Warning states)
- **Info**: `#3b82f6` (Blue - Info messages)

## Icon System

### Standard Icons (24px)
All icons use Lucide React style with 2px stroke width. Icons are referenced as SVG for maximum email client compatibility.

- **Checkmark**: `<polyline points="20 6 9 17 4 12"/>` - Success state (#10b981)
- **Alert Circle**: `<circle>` + `<line>` - Error state (#ef4444)
- **Calendar**: `<rect>` + `<line>` grid - Schedule/dates (#795cf0)
- **Map Pin**: `<path>` + `<circle>` - Places (#ff8e4d)
- **User**: `<path>` + `<circle>` - Users (#999)
- **Book**: `<path>` rectangles - Courses (#795cf0)
- **Mic/CPU**: `<path>` microphone - AI/Intelligence (#795cf0)
- **Bell**: `<path>` bell shape - Notifications (#ff8e4d)
- **Star**: `<polygon>` - Achievements (#fbbf24)
- **Zap**: `<polygon>` lightning - Fast/Priority (#ff8e4d)
- **Music**: `<path>` + `<circle>` notes - Events (#795cf0)
- **Briefcase**: `<rect>` + `<path>` - Professional events (#ff8e4d)
- **Lightbulb**: `<circle>` + `<line>` rays - Tips/Ideas (#795cf0)

### Status Indicators (16px)
- **New**: Badge with "NEW" (#795cf0 bg)
- **Urgent**: Red circle (#ef4444)
- **Complete**: Green checkmark (#10b981)

### Icon Usage Guidelines
- Icons should be inline SVG for maximum compatibility
- Use `stroke="currentColor"` to inherit text color
- Standard stroke-width: 2px
- Always include proper `viewBox="0 0 24 24"` attribute
- Include `fill="none"` for stroke-based icons
- Add `stroke-linecap="round"` and `stroke-linejoin="round"` for smoother appearance

### Example Icon Implementation
```html
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" 
     stroke="currentColor" stroke-width="2" 
     stroke-linecap="round" stroke-linejoin="round">
    <rect x="3" y="4" width="18" height="18" rx="2" ry="2"/>
    <line x1="16" y1="2" x2="16" y2="6"/>
    <line x1="8" y1="2" x2="8" y2="6"/>
    <line x1="3" y1="10" x2="21" y2="10"/>
</svg>
```

- Body text (#f5f5f5) on dark (#1a1a1a): 15.3:1 âœ“
- Headers (#ffffff) on dark (#1a1a1a): 17.8:1 âœ“
- Primary button (#795cf0): Ensure text contrast 4.5:1+
- Links (#795cf0) on dark: 7.2:1 âœ“

### Interactive Elements
- Min touch target: 44x44px
- Focus indicators: 2px solid #795cf0 outline
- Alt text for all images
- Semantic HTML structure

### Dark Mode Native
- All emails designed dark-first
- No mode switching needed
- Optimized for OLED displays
