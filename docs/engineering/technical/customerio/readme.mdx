## Key Concepts

### Message Types
- **Transactional** - DayPlan briefings, notifications
- **Marketing** - Onboarding sequences
- **Push** - Mobile notifications

### Integration Points
- Engine workflows trigger events
- API Router sends transactional messages
- Templates use Liquid for personalization
