## Key Tables

### Core Tables
- `contexts` - Hierarchical entity storage (cities, campuses, courses, students)
- `service_data` - Time-series external data (partitioned by month)
- `accounts` - User account records
- `enrollments` - Course enrollment mappings

### Key Patterns
- **RLS** via `dormway_api` role
- **Partitioning** on `service_data.fetched_at`
- **PostGIS** for geospatial queries (`location` column)
