# Bulk Syllabus Manifest Example

This example mirrors the JSON manifest consumed by `services/engine/src/scripts/bulk-syllabus-import.ts`. Use it as a template when generating batch syllabus uploads prior to running the script.

```json
[
  {
    "file": "./catalog/umich/ENTR-490.pdf",
    "campus": "University of Michigan",
    "campusDomain": "umich.edu",
    "courseCodeHint": "ENTR 490",
    "courseNameHint": "Entrepreneurship Seminar",
    "term": "Fall 2025",
    "year": 2025,
    "instructor": "Prof. Jane Doe"
  },
  {
    "file": "./catalog/umich/EECS-280.pdf",
    "campus": "University of Michigan",
    "campusDomain": "umich.edu",
    "courseCodeHint": "EECS 280",
    "courseNameHint": "Programming and Data Structures"
  }
]
```

> Store the manifest locally, then execute `doppler run --project dormway --config dev_ethan -- node services/engine/dist/scripts/bulk-syllabus-import.js --manifest ./manifest.json` (adjusting paths as needed).
