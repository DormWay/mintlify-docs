## Key Concepts

### Widget Envelope Pattern
Widgets use a standardized envelope structure:
```typescript
interface WidgetEnvelope {
  id: string;      // e.g., "widget:weather"
  type: string;    // e.g., "weather", "schedule"
  content: {
    displayType?: string;
    data: any;
  };
}
```

### Widget Types
- `weather` - Weather display
- `schedule` - Timeline events
- `academic` - Assignments/deadlines
- `dayplan` - Task bank
