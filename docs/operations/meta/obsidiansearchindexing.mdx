# Obsidian Search & Indexing Tools

**Purpose**: Tools to help AI assistants and developers search Obsidian vault using Obsidian's native link format.

---

## Tools Created

### 1. `scripts/build-obsidian-index.sh`

Builds a searchable index of all Obsidian vault files with Obsidian link format.

**Usage**:
```bash
./scripts/build-obsidian-index.sh
```

**Output**:
- `.workspace/obsidian-index.txt` - Text index (always created)
- `.workspace/obsidian-index.json` - JSON index (if `jq` available)

**Index Format**:
```
OBSIDIAN_LINK|RELATIVE_PATH|FILE_NAME|DIRECTORY|SIZE|LINES|MODIFIED
[DEPRECATION_INDEX](/docs/operations/meta/deprecationindex)|Operations/Meta/DEPRECATION_INDEX.md|DEPRECATION_INDEX|Operations/Meta|...
```

### 2. `scripts/search-obsidian.sh`

Search Obsidian vault using Obsidian link format or text search.

**Usage**:
```bash
# Search by Obsidian link
./scripts/search-obsidian.sh "[DEPRECATION_INDEX](/docs/operations/meta/deprecationindex)"

# Text search
./scripts/search-obsidian.sh "Aurora"
./scripts/search-obsidian.sh "widget types"
```

**Output**: Returns Obsidian link format and relative paths for matching files.

### 3. Updated `scripts/export-obsidian-vault.sh`

Now exports files with Obsidian-style paths and link format.

**Changes**:
- Exports use relative paths: `Operations/Meta/file.md` instead of `obsidian-vault/DormWay/Operations/Meta/file.md`
- Includes `OBSIDIAN_LINK` field: `Operations/Meta/file`
- Matches Obsidian's internal file reference format

---

## Why This Matters

### Problem
- AI tools search using filesystem paths: `obsidian-vault/DormWay/...`
- Obsidian uses relative paths: `Operations/...`
- Obsidian links use format: `...`
- Mismatch causes confusion when referencing files

### Solution
- Export script now outputs Obsidian-style paths
- Index maps Obsidian links to actual file paths
- Search tool can find files using Obsidian link format
- AI can reference files using Obsidian's native format

---

## Integration with AI Assistants

When AI assistants search the Obsidian vault:

1. **Use Obsidian link format** when referencing files:
   - ✅ `[DEPRECATION_INDEX](/docs/operations/meta/deprecationindex)`
   - ❌ `obsidian-vault/DormWay/Operations/Meta/DEPRECATION_INDEX.md`

2. **Search using the index**:
   ```bash
   ./scripts/search-obsidian.sh "Link Name"
   ```

3. **Export includes Obsidian format**:
   - Exported files have `OBSIDIAN_LINK` field
   - Paths match Obsidian's internal format

## File Path Formats

| Format | Example | Use Case |
|--------|---------|----------|
| **Filesystem** | `obsidian-vault/DormWay/Operations/Meta/file.md` | Direct file access |
| **Relative** | `Operations/Meta/file.md` | Export/display |
| **Obsidian Link** | `Operations/Meta/file` | Cross-references, search |

## Maintenance

**When to rebuild index**:
- After adding many new files
- After major vault reorganization
- Before running large searches

**Auto-rebuild**:
The search script automatically rebuilds the index if it doesn't exist.
