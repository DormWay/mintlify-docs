# Submodule Setup Guide

The DormWay Obsidian vault is now managed as a Git submodule to enable better collaboration and independent versioning.

## ğŸ¯ What Changed

The vault was moved from a regular folder to a Git submodule:
- **Old location**: `dormway/obsidian/DormWay/` (part of main repo)
- **New structure**: `dormway/obsidian/DormWay/` (submodule pointing to `docs-obsidian` repo)
- **Repository**: `git@github.com:DormWay/docs-obsidian.git`

## ğŸ“‹ For Existing Team Members

If you already have the dormway-dev-tools repo cloned:

```bash
# Update your main repo
cd ~/Development/GitHub/dormway-dev-tools
git pull

# Initialize the submodule
git submodule update --init --recursive

# The vault is now at dormway/obsidian/DormWay
```

## ğŸ†• For New Team Members

```bash
# Clone with submodules
git clone --recurse-submodules git@github.com:DormWay/dormway-dev-tools.git

# Or if already cloned without submodules
cd dormway-dev-tools
git submodule update --init --recursive
```

## ğŸ’¾ Making Changes to the Vault

The vault is a separate repository, so commits work differently:

### 1. Make your changes in Obsidian
Edit notes, add new documents, etc.

### 2. Commit changes to the vault
```bash
cd dormway/obsidian/DormWay
git add .
git commit -m "Your commit message"
git push
```

### 3. Update the parent repo reference
```bash
cd ../../.. # Back to dormway-dev-tools root
git add dormway/obsidian/DormWay
git commit -m "Update obsidian vault reference"
git push
```

## ğŸ”„ Pulling Updates

To get the latest vault changes:

```bash
# From dormway-dev-tools root
git pull
git submodule update --remote

# Or pull directly in the vault
cd dormway/obsidian/DormWay
git pull
```

## ğŸ¯ Why Submodules?

1. **Independent versioning**: Documentation can be versioned separately
2. **Clean history**: Vault changes don't clutter main repo history
3. **Flexible access**: Can grant vault access without full repo access
4. **Better performance**: Large documentation updates don't affect main repo size

## âš ï¸ Common Issues

### "fatal: not a git repository"
You're not in the vault directory. Make sure to:
```bash
cd dormway/obsidian/DormWay
```

### "Changes not staged for commit: modified: DormWay (new commits)"
The submodule has updates. In the main repo:
```bash
git add dormway/obsidian/DormWay
git commit -m "Update vault reference"
```

### Obsidian Git plugin not working
1. Make sure you're in the vault directory
2. Check remote: `git remote -v`
3. Should show: `origin git@github.com:DormWay/docs-obsidian.git`
