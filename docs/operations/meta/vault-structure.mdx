## ğŸ§© Recommended Plugins

### Essential Plugins

1. **Dataview** - Dynamic queries and tables
   - Create automatic indexes
   - Query by tags, dates, or metadata
   - Example: Show all #bug notes modified this week

2. **Templater** - Advanced templates
   - Daily note templates with date/weather
   - Feature templates with standard sections
   - Meeting notes with attendees

3. **Calendar** - Visual daily notes
   - See notes on calendar
   - Quick daily note creation
   - Week/month views

4. **Obsidian Git** - Version control
   - Auto-commit changes
   - Sync across machines
   - Backup to GitHub

### Productivity Plugins

5. **Kanban** - Task boards in Obsidian
   - Visual TODO management
   - Drag-and-drop tasks
   - Links to related notes

6. **Excalidraw** - Sketching and diagrams
   - Architecture diagrams
   - UI mockups
   - Mind maps

7. **Tasks** - Advanced task management
   - Due dates and reminders
   - Task queries across vault
   - Recurring tasks

### Navigation Plugins

8. **Breadcrumbs** - Hierarchy visualization
   - Define parent/child relationships
   - Alternative hierarchies
   - Trail navigation

9. **Quick Switcher++** - Enhanced search
   - Search by heading
   - Search in specific folders
   - Recent files

10. **Waypoint** - Folder notes
    - Automatic folder indexes
    - MOCs (Maps of Content)

## ğŸ” Using the Structure

### Benefits of Hierarchy + Links

1. **Best of Both Worlds**
   - Folders for high-level organization
   - Links for cross-cutting relationships
   - Tags for additional categorization

2. **Multiple Navigation Paths**
   - Browse by folder (temporal, functional)
   - Follow links (conceptual)
   - Search globally (keywords)
   - Query with Dataview (metadata)

3. **Flexible Organization**
   - Move notes between folders easily
   - Links remain intact
   - Can belong to multiple contexts via links
