# Quick Setup Script

This page contains a checklist and instructions for quickly setting up all required plugins for the DormWay vault.

> **Note**: The vault is now managed as a git submodule. Make sure to run `git submodule update --init --recursive` after cloning the main repo.

### Step 3: Enable All Plugins
After installation, make sure all plugins are toggled ON in the Community plugins settings.

### Step 4: Restart Obsidian
Close and reopen Obsidian to ensure all plugin configurations load properly.

### Step 5: Verify Configuration
1. Check that Templates folder exists
2. Try creating a new Daily Note (should use template)
3. Open the Calendar view (should be in left sidebar)
4. Check Git settings (should auto-backup every 10 minutes)

## üîß Manual Configuration Needed

### Git Authentication
1. Go to **Obsidian Git** settings
2. Set up authentication method:
   - **Option A**: Use system Git credentials
   - **Option B**: Configure personal access token
   - **Option C**: Use SSH key

### Template Hotkeys (Optional)
1. Go to Settings ‚Üí Hotkeys
2. Search for "Templater"
3. Set hotkeys for:
   - Insert template: `Cmd/Ctrl + T`
   - Create new from template: `Cmd/Ctrl + Shift + T`

## üÜò Troubleshooting

### Plugin settings not loading?
```bash
# From vault root
git pull
# Then restart Obsidian
```

### Git not working?
1. Check Git is installed: `git --version`
2. Check authentication: `git remote -v`
3. Try manual commit: `git add . && git commit -m "test"`
4. Make sure you're in the right directory: `cd dormway/obsidian/DormWay`
5. The vault is a submodule, so commits go to the docs-obsidian repo

### Templates not applying?
1. Check Templater settings
2. Verify template folder is "Templates"
3. Check folder template mappings
