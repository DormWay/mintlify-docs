# Vault Health Dashboard

This dashboard uses Dataview to show real-time statistics about the vault.

## ‚úÖ Link Health Policy

- **Entry scope must be clean** at all times (`Index.md`, bucket READMEs, `Generated/`).
- **Canonical scope can have missing links only if tracked** in `DormWay/Operations/Meta/Intentional Missing Links.md`.
- **New placeholder rule**: if a link is intentional, log it in the intentional tracker instead of creating a stub.

## üìä Vault Statistics

### Total Notes
```dataview
TABLE WITHOUT ID
  length(file.name) as "Total Notes"
```

### Notes by Type
```dataview
TABLE WITHOUT ID
  length(filter(file.tags, (t) => contains(t, "daily"))) as "Daily Notes",
  length(filter(file.tags, (t) => contains(t, "feature"))) as "Features",
  length(filter(file.tags, (t) => contains(t, "planning"))) as "Planning",
  length(filter(file.tags, (t) => contains(t, "architecture"))) as "Architecture"
```

## üìÖ Recent Activity

### Last 5 Modified Notes
```dataview
TABLE 
  file.mtime as "Modified"
FROM ""
SORT file.mtime DESC
LIMIT 5
```

## üéØ Active Features

### Features by Status
```dataview
TABLE WITHOUT ID
  feature as Feature,
  status as Status,
  priority as Priority,
  owner as Owner
FROM #feature
WHERE status != "completed"
SORT priority ASC
```

## ‚úÖ Open Tasks

### High Priority Tasks
```dataview
TASK
FROM ""
WHERE !completed AND contains(text, "high")
LIMIT 10
```

### Tasks Due This Week
```dataview
TASK
FROM ""
WHERE !completed AND due <= date(today) + dur(7 days)
```

## üîó Orphaned Notes

Notes with no incoming or outgoing links:
```dataview
LIST
FROM ""
WHERE length(file.inlinks) = 0 AND length(file.outlinks) = 0
LIMIT 10
```

## üìà Growth Over Time

### Notes Created This Month
```dataview
TABLE WITHOUT ID
  dateformat(file.ctime, "yyyy-MM-dd") as Date,
  file.name as "Note Created"
FROM ""
WHERE file.ctime >= date(today) - dur(30 days)
SORT file.ctime ASC
```

## üè∑Ô∏è Tag Cloud

### Most Used Tags
```dataview
TABLE WITHOUT ID
  tag as Tag,
  length(rows) as Count
FROM ""
FLATTEN file.tags as tag
GROUP BY tag
SORT length(rows) DESC
LIMIT 20
```
